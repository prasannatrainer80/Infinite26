There are 2 types of exceptions known as 

	(i) Checked Exceptions : These are the exceptions that which occurs at compile time, during compilation JVM feels some erroneous code is there and stops the execution at compile time, can be handled at method level by using throws keyword. 

	(ii) Unchecked Exceptions : These are the exceptions that which occurs at runtime, can be handled by using try...catch block. 

throws : This keyword will be at method level, used to ignore Exceptions at the time of coding. 

throw : This keyword allows you to raise exception objects 
based on the condition.

Write a Program to accept a number if number < 0 then throw ArithmeticException, if number = 0 then throw ArrayIndexOutOfBoundsException 

Custom Exceptions : System Exceptions inadequate to handle in all situations, to overcome this we need the help of custom exceptions. 

Any class that which derived from Exception class can be treated as custom exception. 

syntax : 
________

public class CustomException extends Exception {
	CustomException(String error) {
	   super(error);
	}
}

Files : A file is a collection of data, that which allows you to store output permanently. To deal with files, we need the help of java.io package. 

File class : This class allows you to deal with files. 

File obj = new File("filename");

FileReader : Used to read data char/char and return the ascii value, it returns -1 once it reaches the eof. It reads only text files. 

read() -> Used to read data char/char and returns ascii value.

FileWriter : Used to write data char/char into file. 

write() -> Used to write data char/char.

FileInputStream : used to process data w.r.t. Stream of Bytes. 

FileInputStream fin = new FileInputStream(filename);

read() : used to read data char/char

FileOutputStream : Used to write data w.r.t. Stream of byte

DataOutputStream : Used to write Primitive Types into the file

writeInt()
writeUTF()
writeDouble()
writeBoolean()

DataInputStream : Used to read primitive types from the file. 

readInt()
readUTF()
readDouble()
readBoolean()

BufferedReader : Allows you to read input from the keyboard. 

BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

readLine() : used to read one line of input. 

ObjectOutputStream : used to write Objects into the file.

writeObject() : this method used to write the objects into the file.

ObjectInputStream : Used to read objects from the file.

readObject() : this method used to read object from the file.

Serilization : It is a process of writing the custom objects into the file. Any class objects to be stored in the file must be implemented with Serilizable interface.
